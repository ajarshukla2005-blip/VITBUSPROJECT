# VITBUS : College Bus Schedule & Attendance HelpeR
## Description

In our college, it’s always confusing to know which bus to take in the morning or evening and which stop to stand at. Teachers also keep reminding us about attendance issues. So, I made this Python project where you can:

- Register and login as yourself
- See bus routes, their timings, and even search routes by your bus stop
- Mark your attendance for each day and check your summary with percentage

I used only the basic Python stuff (so everyone can run it without trouble) and all data’s saved in small JSON files in a `data/` folder.
